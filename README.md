Auctionaire is a simple yet functional **Java-based auction system** designed to manage item listings, bidding, and user interactions efficiently.  
This project demonstrates core concepts in **Java OOP, GUI components, event handling, and basic server-client logic**.

---

### Features  
-  Add new auction items  
-  Place bids on available items  
-  Automatic update of highest bid  
-  Admin panel using Java Swing  
-  Server-side logic to manage bids  
-  Basic HTML interface (Index.html)
